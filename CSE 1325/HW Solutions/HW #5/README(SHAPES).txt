File included: 
makefile 
shapes.cpp 
shapes.h 
shapes_main.cpp

**************
The above file compiles and runs on a Gtk window. 
User can click on the shapes they like and it will show 
the formula to calculate perimeter and area of that shape.
User can find out the total number of shapes they clicked before exiting the program.
**************
Compilation instructions:
->This is built in a Lubuntu virtual machine
->In terminal:
	make
	./shapes
